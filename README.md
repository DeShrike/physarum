# Physarum

WIP

## Quick start

Clone this repo.

Install dependencies, see below.

Then do:

```console
cd physarum
make
./physarum
```

## Install dependencies

```console
sudo apt-get install libsdl2-dev
sudo apt-get install libsdl2-image-dev
sudo apt-get install libsdl2-ttf-dev
```
